METHOD-1
========
1. docker ps -a --format "{{.Status}} {{.Port}}"

2. docker ps -a --format "{{.ID}} {{.Command}}"

3. docker ps -a --format "{{.Container}} {{.CreatedAt}}"

4. docker ps -a --format "{{.ID}} {{.Names}}"

5. docker ps -a --format "{{.ID}}"


METHOD-2
========

Correct ways to filter by container:
a) Filter by container name:
docker ps -a --filter "name=my-container" -q

b) Filter by container ID (partial ID works too):
docker ps -a --filter "id=91b160bab7ea" -q

c) Filter by status:
docker ps -a --filter "status=exited" -q

docker ps -a --filter "status=running" -q------------>running container ID

🧱 1️⃣ Filter by Container ID
=============================
✅ Command:
bash
Copy code
docker ps -a --filter "id=<container_id>"
Example:
bash
Copy code
docker ps -a --filter "id=ce72a5e6825e"
🟢 Shows only that container.

You can also use partial ID:

bash
Copy code
docker ps -a --filter "id=ce72"


🏷️ 2️⃣ Filter by Image (Repository + Tag)
=========================================
✅ Command:
bash
Copy code
docker ps -a --filter "ancestor=<image_name>"
Example:
bash
Copy code
docker ps -a --filter "ancestor=sonarqube:latest"
🟢 Shows all containers created from that image.
You can use partial image name like ancestor=sonarqube.


📦 3️⃣ Filter by Container Name
==============================
✅ Command:
bash
Copy code
docker ps -a --filter "name=<name>"
Example:
bash
Copy code
docker ps -a --filter "name=nexus"
🟢 Shows all containers where the name matches (even partial names).


⏰ 4️⃣ Filter by Creation Time (Before / Since)
===============================================
✅ Command:
bash
Copy code
docker ps -a --filter "before=<container_name_or_id>"
→ Shows containers created before that container.

bash
Copy code
docker ps -a --filter "since=<container_name_or_id>"
→ Shows containers created after that container.

Example:
bash
Copy code
docker ps -a --filter "before=sonarqube"
docker ps -a --filter "since=nexus"


⚙️ 5️⃣ Filter by Status
=====================
✅ Command:
bash
Copy code
docker ps -a --filter "status=<state>"
States:
created

restarting

running

paused

exited

dead

Example:
bash
Copy code
docker ps -a --filter "status=exited"
docker ps -a --filter "status=running"
🟢 Lists containers in that state.


🔢 6️⃣ Filter by Exit Code
=========================
✅ Command:
bash
Copy code
docker ps -a --filter "exited=<code>"
Example:
bash
Copy code
docker ps -a --filter "exited=0"
🟢 Shows containers that exited successfully (exit code 0).


🔗 7️⃣ Filter by Network
=======================
✅ Command:
bash
Copy code
docker ps -a --filter "network=<network_name>"
Example:
bash
Copy code
docker ps -a --filter "network=bridge"
🟢 Shows all containers connected to the bridge network.


📁 8️⃣ Filter by Volume
========================
✅ Command:
bash
Copy code
docker ps -a --filter "volume=<volume_name>"
Example:
bash
Copy code
docker ps -a --filter "volume=nexus-data"
🟢 Shows containers using that volume.


🏷️ 9️⃣ Filter by Label
=======================
✅ Command:
bash
Copy code
docker ps -a --filter "label=<key=value>"
Example:
bash
Copy code
docker ps -a --filter "label=maintainer=anil"
🟢 Shows containers built or run with that label.


🧩 🔟 Combine Multiple Filters
===============================
✅ Command:
bash
Copy code
docker ps -a \
  --filter "status=exited" \
  --filter "ancestor=ubuntu" \
  --filter "name=test"
🟢 Shows all exited containers from ubuntu image with name containing test.
